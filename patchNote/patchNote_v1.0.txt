* 공통 패치
- 레벨당 직업코인 획득 1개 -> 3개 (조정)
- 직업 최대 2개선택 (코인으로 추가 직업 선택 불가)


* 연금술사 레벨업
- 물약을 양조하면 직업경험치 1~2증가 => 직업경험치 1 고정 (너프)
- 물약을 마시면 직업경험치 1~2증가 => 직업경험치 2~4 (버프)
- 물약투척시 직업경험치 1~2증가 => 직업경험치 2~4 (버프)
- 네더사마귀 수확시 직업경험치 1~2증가 => 직업경험치 1 고정 (너프)
- 해당 엔티티 처치시 직업경험치 1~2증가 => 직업경험치 2~3 (버프)
