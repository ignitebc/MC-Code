* 연금술사 레벨업

- 물약을 마시면 직업경험치 1~2증가 => 직업경험치 2~4 (버프)
- 물약투척시 직업경험치 1~2증가 => 직업경험치 2~4 (버프)
- 네더사마귀 수확시 직업경험치 1~2증가 => 직업경험치 1 고정 (너프)
- 해당 엔티티 처치시 직업경험치 1~2증가 => 직업경험치 2~3 (버프)

* 굴착공 레벨업 (건축가 경험치 합류)
- 일반 블록 (흙, 모래만) 채굴시, 직업경험치 0 ~ 2 => 직업경험치 0 ~ 1  (너프)
- 희귀 블록	(흙/모래/눈 제외한 삽으로 채굴 가능한 블록 중) 채굴시, 직업경험치 2~4 => 직업경험치 2 고정 (너프)
- 경도가 0.2-2.9 사이 블록설치시 직업경험치 0~2증가 (나무,흙,모래,양털 등) => 직업경험치 0~1 (버프)
- 경도가 3.0-9.9 사이 블록설치시 직업경험치 2~4증가 (돌,벽돌,석재 등) => 직업경험치 1~2 (너프)
- 경도가 10-100 사이 블록설치시 직업경험치 4~6증가 (흑요석, 기반암 등) => 직업경험치 2~4 (너프)

* 농부 레벨업
- 작물 심기 직업경험치 0 ~ 1 => 직업경험치 1 고정 (버프)
- 작물 수확 (완전히 자란 것) 직업경험치 1 ~ 3 => 직업경험치 0 ~ 2 (너프)
- 양 털 깎기 직업경험치 2 ~ 5	=> 직업경험치 1~3 (너프)
- 동물 번식 직업경험치 6 ~ 10 => 직업경험치 2~5 (너프)
- 동물 길들이기 직업경험치 25 ~ 50 => 직업경험치 4~8 (너프)
- 수영 할때 (10블록이상) 직업경험치 1 ~ 2 => 직업경험치 1 고정 (너프)
- 낚시, 직업경험치 5~10 => 직업경험치 4 ~ 6 (너프)

- 해당 특수아이템 낚을시 직업경험치 10~20 => 직업경험치 10 고정 (너프)

* 사냥꾼 레벨업
- 아이템을 제련/구울때 직업경험치 2~4 (화로나 용광로가 아닌 훈연기에서 구운경우만 적용) => 직업경험치 4~6 (버프)
- 모든 몬스터/동물 처치시 직업경헝치 4~6 => 직업경험치 2~4 (너프)


* 광부 레벨업
- 일반블록 (돌,조약돌,화강암,안산암,섬록암,심층암,네더랙,엔드스톤) 채굴시 직업경험치 0~2 => 직업경험치 0~1 고정 (너프)
- 희귀블록 (벽돌, 석재, 네더 벽돌 등) 채굴시 직업경험치 2~4, 직업경험치 1~2 고정 (너프)
- 일반광석 (철, 금, 석탄, 레드스톤 등) 채굴시 직업경험치 4~6, 직업경험치 4 고정 (너프)
- 희귀광석 (다이아몬드, 에메랄드, 고대 잔해) 채굴시 직업경험치 6~8, 직업경험치 6 고정 (너프)


* 대장장이 레벨업
- 광석 제련 시 직업경험치 2~4 => 직업경험치 4~6 (버프)
- 모루 사용 시 직업경험치 4~6 => 직업경험치 5~10 (버프)
- 숯돌 사용 시 직업경험치 4~6 => 직업경험치 5~10 (버프)
- 희귀광석 (다이아몬드, 에메랄드, 고대 잔해, 레드스톤, 청금석) 채굴시 직업경험치 4~6 (추가)
- 인챈터 작업대, 첫 번째 슬롯(가장 위)에서 인챈트 적용시 직업경험치 30~50 => 직업경험치 10 고정 (추가, 조정)
- 인챈터 작업대, 두 번째 슬롯(가운데)에서 인챈트 적용시 직업경험치 50~70 => 직업경험치 20 고정 (추가, 조정)
- 인챈터 작업대, 세 번째 슬롯(가장 아래)에서 인챈트 적용시 직업경험치 70~100 => 직업경험치 50 고정 (추가, 조정)
